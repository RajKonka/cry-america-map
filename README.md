# CRY America — Interactive Project Map

Interactive map showing all 40 CRY America project locations across India.

**Live URL:** [https://rajkonka.github.io/cry-america-map/](https://rajkonka.github.io/cry-america-map/)

## Features
- Zoomable India map with proper state borders (Leaflet + CartoDB tiles)
- 40 project pins — click any for details and a link to the project page
- Funding goal and progress shown where available
- Fully responsive — works on desktop and mobile

## Setup
This is a single `index.html` file. No build step needed.

1. Clone this repo
2. Enable GitHub Pages (Settings → Pages → Source: `main` branch, `/ (root)`)
3. Visit `https://rajkonka.github.io/cry-america-map/`

## Usage in PowerPoint
The accompanying `.pptx` file has a clickable **"Open Interactive Map"** button that links to the hosted URL. In presentation mode, clicking it opens this map in the browser.

Alternatively, in PowerPoint 365:
1. Go to **Insert → Add-ins → Web Viewer**
2. Paste the GitHub Pages URL
3. The live map renders inside the slide

## Data
Project data sourced from [cryamerica.org/projects](https://www.cryamerica.org/projects/).
